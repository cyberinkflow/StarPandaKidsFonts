# ⭐️ 星光小熊貓幼兒體－星光敘事，熊貓寫字。
👉 「星光小熊貓幼兒體」是白河透過 zi-hi.com 自製的第一個手寫字型。

初始風格設定是比較可愛又有點顫抖的小學生體筆觸，但是寫著寫著手越來越痠就越來越……隨緣佛系。

目前已經完成5500+，剩下的字會慢慢在手不痠之後陸續寫完釋出。

目前測試：用來寫甄嬛傳台詞、周星馳台詞、甚至部份佛經，以及工作的腳本稿件都沒有缺字，可供日常使用。
